# Music Player

A sleek, ad-free music player that sources audio from YouTube. Features intelligent playlist management, dynamic theming, and multiple playback modes.

## Features

- **Search**: Search for songs on YouTube
- **Playback Modes**: Order, Once, Repeat x2, Repeat All, Shuffle
- **Auto-suggestions**: Smart song suggestions
- **Playlists**: 
  - Favorites: Save your favorite songs
  - History: Songs played in session
  - Top Played: Most frequently played songs
- **Controls**: Play/Pause, Next/Previous, Volume, Progress bar with seek
- **Themes**: Light, Dark, and Dynamic (adapts to album art)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Howler.js for audio playback
- Lucide React for icons

## Note

This app uses YouTube for searching and retrieving audio. Please respect YouTube's terms of service when using this application.
